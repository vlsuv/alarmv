# Alarmv
Simple alarm application for ios.

# Features
- Model-View-Controller architecture.
- Support for light & dark mode.
- Data persistence using Core Data.
- Local Notifications.
- Use UserDefaults for saving dark mode status.
- Fully programmatically UI.
